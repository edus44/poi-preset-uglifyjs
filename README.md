# poi-preset-yaml

Add YAML support

## Install

```bash
npm i -D poi-preset-yaml
yarn add poi-preset-yaml --dev
```

## Usage

Activate it in config file `poi.config.js`:

```js
module.exports = {
  presets: [require('poi-preset-yaml')()],
}
```
